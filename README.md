# Smart DSA Visualizer

Smart DSA Visualizer is a web-based platform that helps students understand
Data Structures and Algorithms through interactive visualizations.

## 🚀 Live Demo
https://smart-dsa-visualizer-vu4r.onrender.com

## 📚 Features
- Sorting Algorithm Visualization
- Stack Operations
- Queue Operations
- BFS & DFS Graph Traversal
- Dijkstra Shortest Path
- A* Algorithm
- Binary Search
- Linked List Visualization
- Tree Structures
- Hash Tables

## 🛠️ Technologies Used
- Python
- Django
- HTML
- CSS
- JavaScript
- Bootstrap

## 📦 Installation

Clone the repository:

git clone https://github.com/saisurya-2406/smart-dsa-visualizer.git

Run the project:

python manage.py runserver
