Algorithm Details

Bubble Sort: Repeatedly swaps adjacent elements if in wrong order. Simple but inefficient. O(n²).

Insertion Sort: Builds sorted array by inserting elements one by one. Good for small arrays. O(n²).

Quick Sort: Divides array into partitions, recursively sorts. Fast average case. O(n log n).

Merge Sort: Divides into halves, sorts and merges. Stable and efficient. O(n log n).
